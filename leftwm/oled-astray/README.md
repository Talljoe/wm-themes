# OLED Astray

A high-contrast theme for OLED Displays based on the stunning images by [h heyerlein](https://unsplash.com/@heyerlein?utm_medium=referral&amp;utm_campaign=photographer-credit).

## Requirements

* Kitty
* Polybar
* Compton
* LeftWM
* Fish
* Font Awesome 5
* Fira Code
